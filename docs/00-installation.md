# Instalasi

Untuk mengaktifkan library Ideahut Quarkus di project.

## Maven

``` xml
<dependency>
	<groupId>net.ideahut</groupId>
	<artifactId>ideahut-quarkus-3x</artifactId>
	<version>26.04.04</version>
</dependency>
<dependency>
	<groupId>net.ideahut</groupId>
	<artifactId>ideahut-quarkus-3.3x.2</artifactId>
	<version>26.04.04</version>
</dependency>

```

## Gradle

``` cmd
implementation group: 'net.ideahut', name: 'ideahut-quarkus-3x', version: '26.04.04'
implementation group: 'net.ideahut', name: 'ideahut-quarkus-3.3x.2', version: '26.04.04'
```

## Versi
Berikut adalah tabel artifact yang digunakan disetiap versi quarkus. <br/>
Aplikasi bisa error jika artifact yang digunakan tidak sesuai dengan versi quarkus.
|Artifact|Quarkus|
|:---------:|:----------:|
|__ideahut-quarkus-3x__|<small>(wajib di semua quarkus versi 3)</small>|
|ideahut-quarkus-3.3x.1|__3.30.0 - 3.31.4__|
|ideahut-quarkus-3.3x.2|__3.32.0 - Terbaru__|
